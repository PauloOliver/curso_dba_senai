Modelagem Física: Escrevendo os Scripts CREATE TABLE

Agora, traduzimos nosso Modelo Lógico finalizado e normalizado para comandos SQL DDL que o SGBD (MySQL) entende.

Processo Guiado:

  1. Criar a tabela Editoras.
  2. Criar a tabela Autores.
  3. Criar a tabela Generos.
  4. Criar a tabela Livros (com FK para Editoras).
  5. Criar a tabela associativa Livro_Autores (com FKs para Livros e Autores).
  6. Criar a tabela associativa Livro_Generos (com FKs para Livros e Generos).
  7. Criar a tabela Membros.
  8. Criar a tabela Emprestimos (com FKs para Livros e Membros).

Foco na sintaxe MySQL:

  • INT AUTO_INCREMENT PRIMARY KEY
  • VARCHAR(n) NOT NULL
  • UNIQUE
  • DECIMAL(p,s)
  • DATE, TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  • FOREIGN KEY (col) REFERENCES tabela_pai(col_pai) [ON DELETE ...]
  • CONSTRAINT nome_constraint CHECK (condicao)

Vamos usar o DER final como nossa referência visual constante.
