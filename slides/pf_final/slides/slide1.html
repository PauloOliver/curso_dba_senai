Exercício Preparatório – O projeto final?

Este exercício tem como objetivo preparar você para a prova final da disciplina de Banco de Dados.

Trata-se de uma versão simplificada do trabalho completo, abordando as mesmas etapas de forma reduzida.

Tema sugerido: Sistema de Controle de Biblioteca Escolar

1. Modelagem Conceitual

  • Monte um DER (Diagrama Entidade-Relacionamento) com 4 entidades: Aluno, Livro, Autor e Empréstimo.
  • Defina as chaves primárias e as cardinalidades (1:N, N:M, etc.).
  • Certifique-se de que o modelo representa a lógica real do sistema proposto.

2. Modelagem Lógica

  • Converta o DER em tabelas relacionais com todos os atributos necessários.
  • Defina chaves primárias, estrangeiras e restrições como NOT NULL, UNIQUE e CHECK.
  • Se necessário, crie tabelas auxiliares e aplique a 3ª Forma Normal (3FN).

3. Desenvolvimento do Banco de Dados (SQL)

  • Crie dois scripts SQL:
    • DDL: criação das tabelas e relacionamentos.
    • DML: inserção de dados de exemplo (mínimo 3 por tabela).
  • Crie 1 Stored Procedure para registrar um empréstimo.
  • Crie 1 Trigger para atualizar o status do livro (ex: “Emprestado”).
  • Crie 1 View para listar empréstimos com nome do aluno e título do livro.

4. Testes e Demonstração

  • Execute consultas que demonstrem o funcionamento do banco:
    • Listar livros e autores.
    • Mostrar livros emprestados e seus respectivos alunos.
    • Testar INSERT, UPDATE e DELETE (verificar integridade referencial).
    • Consultar a View e executar a Stored Procedure.

5. Entrega e Avaliação

  • Enviar um arquivo .sql com todos os scripts (DDL + DML + extras).
  • Incluir o DER (feito no Draw.io, Workbench ou similar).
  • Adicionar capturas de tela das consultas funcionando.
  • A pontuação seguirá os critérios da prova, adaptados para este exercício.
