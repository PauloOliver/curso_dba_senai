DEMO: SP ListarAlunosPorCurso (Parâmetro IN)

Cenário: Criar uma SP que recebe o nome de um curso e retorna os alunos matriculados nele.

Código SQL (MySQL):


DELIMITER $$
CREATE PROCEDURE ListarAlunosPorCurso (IN pCurso VARCHAR(50))
BEGIN
    SELECT id, nome, idade
    FROM Aluno
    WHERE curso = pCurso;
END $$
DELIMITER ;

Chamando e Usando (MySQL):


CALL ListarAlunosPorCurso('Engenharia');

Demonstração ao Vivo: Criar a tabela, inserir dados e executar a SP para ver apenas os alunos de Engenharia.
